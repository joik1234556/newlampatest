-------------------------------------------------------------------------------
ЧАСТЬ 1. КАК РАБОТАЕТ plugin_modss.js
-------------------------------------------------------------------------------

plugin_modss.js — это полноценная «платформа плагинов» для медиаплеера Lampa.
Размер: ~15 000 строк. Версия 3.3.

Основные модули внутри одного файла:
  • Modss.init()         — точка входа, собирает всё вместе
  • Modss.online()       — кнопка «Смотреть онлайн» на странице фильма/сериала
  • Modss.preload()      — предзагрузка источника по нажатию кнопки
  • Modss.tv_modss()     — IPTV-телевидение (компонент modss_tv)
  • Modss.radio()        — радио (компонент Radio_n)
  • Modss.collections()  — коллекции фильмов
  • Modss.auth()         — авторизация через Telegram-бот (@modssmy_bot)
  • Modss.balansers()    — список 23 балансеров (источников онлайн-видео)
  • Modss.check()        — проверка Jackett-сервера
  • Modss.rating_kp_imdb() — загрузка рейтингов KP/IMDB с сервера
  • Modss.serialInfo()   — показывает номер последнего/нового эпизода на постере
  • Modss.Subscr         — подписка на новые серии (push-уведомления)
  • SourceFilmix         — встроенный источник Filmix (интеграция в Lampa.Api)
  • SourcePub            — встроенный источник KinoPub
  • ForkTV               — авторизация через ForkTV
  • modss_tv             — компонент IPTV
  • Radio_n              — компонент радио
  • component            — компонент "modss_online" (балансерный онлайн-просмотр)
  • collection           — компонент коллекций

Поток работы при открытии фильма:
  1. Lampa.Listener.follow('full') → в конце страницы фильма появляется кнопка
     "▶ modss_title_online" с бейджем "VIP".
  2. По нажатию: кнопка меняется на анимированный спиннер, фоном стартует
     Modss.preload() — запрашивает источник через API api.n.modss.tv.
  3. API возвращает ссылки от выбранного балансера (Filmix, HDRezka, Alloha и т.д.)
     без TorBox — сразу готовые HTTP-ссылки.
  4. Если источник нашёл контент: кнопка восстанавливается, долгое нажатие
     открывает выбор перевода/качества.
  5. По нажатию (или hover:enter) открывается полноценный экран балансера
     (компонент modss_online) с постерами, эпизодами, фильтром перевода.

Авторизация:
  Плагин использует VIP-систему:
    uid = 'bb04b81f..._234350575' + ping к api.n.modss.tv/device/auth
    VIP-дни показываются рядом с заголовком кнопки.
    Без авторизации часть балансеров (Mango 4K, Alloha 4K, HDRezka и др.) недоступна.

Jackett:
  В modss плагине Jackett используется как один из дополнительных поисковых
  источников для торрент-парсера. Настройки (URL + API-ключ) задаются
  в разделе Настройки → Парсер. Modss хранит несколько публичных Jackett-серверов
  (jac.red, jacred.torrservera.net, spawn.pp.ua:59117) в объекте Modss.jack.

ЧАСТЬ 2. ДИЗАЙН plugin_modss.js
-------------------------------------------------------------------------------

2.1 CSS — всё встроено в JS
  Modss не использует отдельный .css файл.
  CSS инжектируется прямо в DOM через Lampa.Template.add('modss_online_css', "<style>…</style>").
  Вставка: $('body').append(Lampa.Template.get('modss_online_css', {}, true));

  Основные CSS-классы:
    .online_modss              — карточка источника (flex-строка)
    .online_modss__img         — постер эпизода/фильма (13em × 8.2em, cover)
    .online_modss__body        — текстовый блок справа
    .online_modss__title       — название, усечение через -webkit-line-clamp
    .online_modss__quality     — качество (4K, FHD, 720p)
    .online_modss__timeline    — прогресс просмотра
    .online_modss--full.focus  — при фокусе scale(1.02) + золотистый фон #b58d36
    .online_modss.focus::after — белая рамка 0.3em вокруг карточки при фокусе
    .online-empty              — экран "ничего не найдено" с шаблонными строками
    .modss-balanser-loader     — SVG-спиннер (анимированный круг) в кнопке

  Цветовая палитра:
    • Синий бейдж #168FDF   — "уже просмотрено", "новая серия", "последний сезон"
    • Золотой #ffe216        — VIP-статус, оставшиеся дни
    • Белый #fff             — кнопки, рамки фокуса
    • Тёмный RGBA            — подложки под постерами

2.2 Кнопка на странице фильма
  HTML (упрощённо):
    <div class='full-start__button selector view--modss_online' data-subtitle='modss_v3.3 (23 Balansers, 18 in VIP)'>
      <svg class="modss-online-icon">…</svg>
      <span>modss_title_online 💎 87 дней</span>
      <div style='…background: white; color: black; font-size: 0.6em; border-radius: 0.5em;'>VIP</div>
    </div>

  Особенности:
    • Иконка — SVG "play-стрелка" (кастомная, цвет white)
    • Рядом с названием — количество оставшихся VIP-дней
    • Белый бейдж "VIP" внизу слева кнопки (абсолютное позиционирование)
    • При загрузке текст заменяется на анимированный SVG-спиннер + "Загрузка"
    • При ошибке кнопка бледнеет (opacity 0.3) и надпись возвращается

2.3 Экран балансера (компонент modss_online)
  • Список source/серий отображается как карточки с постерами (как в Lampa native)
  • Каждая карточка: постер слева + название + качество + прогресс-бар просмотра
  • Просмотренные отмечаются иконкой "глаза" поверх постера
  • Фильтры сверху (перевод, сезон) через нативный Lampa.Filter
  • При прокрутке к последнему просмотренному — подсветка синим бейджем

2.4 Регистрация шаблонов
  Lampa.Template.add('modss_online_full', "…html-шаблон…");
  Lampa.Template.add('modss_online_folder', "…");
  Lampa.Template.add('modss_online_rate', "…рейтинг звёздочка…");
  Lampa.Template.add('modss_online_watched', "…иконка воспроизведения…");
  Lampa.Template.add('modss_online_season', "…номер сезона…");
  Lampa.Template.add('modss_styles', "…глобальные стили…");
  Lampa.Template.add('modss_does_not_answer', "…экран ошибки с кнопками…");
  Lampa.Template.add('onlines_v1', "…классический вид строки…");

2.5 Настройки
  Modss добавляет собственный раздел "MODS's 💎" в Настройки Lampa через
  Lampa.SettingsApi.addComponent и Lampa.SettingsApi.addParam.
  Параметры: включение Online, выбор приоритетного балансера, прокси,
  уведомления, пароль, TV, Radio, параметры KinoPub, ForkTV, HDRezka, Filmix.

ЧАСТЬ 4. ЧТО МОЖНО УЛУЧШИТЬ В easy-mod.js, ВЗЯВ ЗА ОБРАЗЕЦ modss
-------------------------------------------------------------------------------

Следующие улучшения из modss-плагина можно применить к easy-mod.js:

4.1 ДИЗАЙН КНОПКИ (приоритет: высокий)
  Что делает modss:
    В кнопке на странице фильма, пока идёт загрузка, отображается
    анимированный SVG-спиннер + надпись "Загрузка". По завершении —
    текст кнопки восстанавливается.
  Что делает easy-mod:
    Нет спиннера в кнопке — открывается новый экран с текстом "Запуск потока…"
  Как улучшить:
    Добавить в injectEasyButtonModssStyle() логику:
      btn.html(loader + '<span>Поиск…</span>') во время запроса /variants
      btn.html(ico + '<span>Easy-Mod</span>') после получения ответа

4.2 КАРТОЧКИ ВАРИАНТОВ С ПОСТЕРАМИ (приоритет: средний)
  Что делает modss:
    Каждый вариант отображается как карточка с постером фильма слева
    (13em × 8.2em) и текстом справа. Используются шаблоны Lampa.Template.
  Что делает easy-mod:
    Простые текстовые строки без постеров.
  Как улучшить:
    Добавить постер из movie.poster_path при отрисовке вариантов:
      '<div class="online_modss__img"><img src="..." /></div>'
    Использовать классы online_modss для единого стиля с Lampa.

4.3 НАТИВНЫЙ SCROLL (приоритет: средний)
  Что делает modss:
    Lampa.Scroll({ }) — нативный скролл с поддержкой пульта/клавиатуры
  Что делает easy-mod:
    Flat div без scroll-контейнера
  Как улучшить:
    Обернуть список вариантов в Lampa.Scroll (как в Modss.online-компоненте)

4.4 ВСТРОЕННЫЙ CSS (вместо внешнего файла)
  Что делает modss:
    Все стили в одном JS-файле — плагин самодостаточен, не нужен доп. файл
  Что делает easy-mod:
    CSS в отдельном easy-mod.css (нужно, чтобы сервер его отдавал)
  Как улучшить (опционально):
    Перенести содержимое easy-mod.css в JS через:
      $('<style>').text('…').appendTo('head')
    Тогда плагин будет работать даже если /static/easy-mod.css недоступен

4.5 АНИМАЦИИ ФОКУСА (приоритет: низкий)
  Что делает modss:
    .focus::after — белая рамка + scale(1.02) для карточек при фокусе
  Что делает easy-mod:
    Наследует стили фокуса от Lampa (нет кастомного scale)

4.6 РАЗДЕЛ НАСТРОЕК (приоритет: средний)
  Что делает modss:
    Полный раздел настроек в Lampa.Settings через Lampa.SettingsApi
  Что делает easy-mod:
    Нет настроек
  Как улучшить:
    Добавить минимальные параметры (URL сервера, таймаут опроса)
    через Lampa.SettingsApi.addComponent / addParam

4.7 ПЕРЕВОДЫ ЧЕРЕЗ Lampa.Lang (приоритет: низкий)
  Что делает modss:
    Все строки через Lampa.Lang.translate('ключ')
  Что делает easy-mod:
    Строки хардкодом по-русски
  Как улучшить:
    Зарегистрировать строки через Lampa.Lang.add({...}) и использовать translate()

4.8 ИНДИКАТОР НА ПОСТЕРЕ (просмотрено/новая серия)
  Что делает modss:
    На постере фильма появляется синий бейдж "S3:E5" с последним просмотренным
    эпизодом, и прогресс-бар просмотра под постером.
  Что делает easy-mod:
    Нет такой функции

КРАТКИЙ ВЫВОД
-------------------------------------------------------------------------------

plugin_modss.js — это «комбайн» из 15 000 строк с 23+ источниками, TV, Радио,
VIP-авторизацией, богатыми карточками с постерами, анимациями и нативными
компонентами Lampa. Он работает с закрытым сервером api.n.modss.tv и требует
подписки.

easy-mod.js — это ~700-строчный точечный плагин для вашего собственного сервера.
Он берёт торренты через Jackett/Torrentio и стримит через TorBox. Дизайн
минималистичный, но функция чёткая: найти торрент → получить прямую ссылку →
воспроизвести.

Главное что взять из modss в easy-mod:
  1. Спиннер внутри кнопки (вместо нового экрана при загрузке)
  2. Карточки вариантов с постером (как .online_modss)
  3. Нативный Lampa.Scroll для списка
  4. CSS встроить в JS (self-contained плагин)
